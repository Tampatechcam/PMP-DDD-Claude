### Findings

⚠️ **Colgroup hydration warning — found and fixed in this session.**
`components/orders/OrdersList.tsx:220-230` had a `<colgroup>` with inline JSX comments (`{/* ... */}`) and conditional `{!isPast && <col />}` expressions. In Next.js RSC streaming, these produce whitespace text nodes that HTML forbids inside `<colgroup>`, causing repeated hydration warnings: `Warning: In HTML, whitespace text nodes cannot be a child of colgroup`. Fix applied: removed `<colgroup>` entirely; moved column widths to `className` props on `<Th>` elements (extended `Th` to accept optional `className`). With `table-fixed`, the `<thead>` widths govern layout — `<colgroup>` was redundant. Warning count: 24 before fix, 0 after.

🔍 Admin invoice Pill verification done via source only — client session cannot reach `/admin/invoices`. The import and render call in source are unambiguous; no live browser proof of the admin invoice view.
